Guia básico de HTML

Para iniciantes (ou não)

HTML (HyperText Markup Language) é uma linguagem de marcação usada para construir páginas para a web.

Mas isso você já sabia.

Ele pode ser usado também para escrever emails, documentos, aplicativos móveis e mais. E o melhor? Você não precisa ser um mestre da programação para começar, a linguagem é super amigável. Muita gente já usa HTML para personalizar a aparência do seu blog no Tumblr, por exemplo.

E falando de personalização, não podemos deixar de citar o CSS e o JS. Vamos falar deles mais pra frente.
